# Zootopia API Generator

Dieses Projekt ermöglicht es Benutzern, Informationen über verschiedene Tierarten von einer API abzurufen und diese automatisch in einer schön gestalteten Webseite darzustellen.

## Funktionen

* Abruf von Tierdaten (Ernährung, Standort, Typ) über die API-Ninjas.
* Dynamische Erstellung einer HTML-Webseite basierend auf Benutzereingaben.
* Fehlerbehandlung, falls ein Tier nicht in der Datenbank existiert.
* Sichere Verwaltung des API-Keys über eine `.env`-Datei.
